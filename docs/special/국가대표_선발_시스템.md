## 1. KFA 유소년 선발 체계 개요

대한축구협회(KFA)는 **4단계 피라미드 구조**로 유소년 선수를 발굴·육성한다.

```
KFA센터 (국가대표급 최상위)
    ↑
합동광역센터
    ↑
광역센터 (5개 — KFA + 시도 협회 공동 운영)
    ↑
지역센터 (21개 — 시도 협회 운영)
```

연령별 프로그램 명칭은 다음과 같다.

| 연령 | 프로그램명 | 비고 |
|------|-----------|------|
| 6~11세 | 프리 골든에이지 (Pre Golden Age) | 기본기 형성기 |
| 12~15세 | 골든에이지 (Golden Age) | 핵심 기술 습득기, U-14 대표팀 선발 대상 |
| 16~19세 | 포스트 골든에이지 (Post Golden Age) | U-17·U-20 연결 단계 |

KFA는 2019년부터 AFC의 '엘리트 유소년 축구 발전 계획' 프로젝트에 참여하여 **3스타 정규 멤버십**을 획득하였으며, 조직·계획·경기·지도자·유망주 발굴·의료·스포츠과학 등 11개 항목에서 평가를 받는다.

## 2. U-14 (골든에이지 대표팀)

KFA는 공식적으로 **U-14 남자 국가대표팀**을 운영한다. (KFA 공식 페이지: `kfa.or.kr/national/?act=wu_14`)

| 항목 | 내용 |
|------|------|
| 주관 | KFA 기술위원회 / 유소년 전임 지도자 |
| 대상 | 만 14세 이하 (중학교 1~2학년 연령대) |
| 선발 방식 | 지역센터·광역센터 추천 → KFA 센터급 합동 훈련 캠프 → 최종 발탁 |
| 선발 기준 | 리그 성적, 신체 능력, 기술 평가, 지도자 추천 |
| 소집 횟수 | 연 2~4회 소집 훈련 (확인 필요 — KFA 공식 일정 기준) |
| 최종 엔트리 | 약 20~25명 (확인 필요) |
| 주요 활동 | 국내 훈련 캠프, 해외 유소년 친선 대회 참가 |
| 국제 대회 | 별도의 FIFA 공인 연령별 대회 없음 — 친선대회 위주 |

**선발 절차 요약**: 지역별로 연령별 등록 선수를 전임 지도자가 관찰 → 광역센터 훈련을 통해 평가 → KFA가 최종 대표팀 명단 구성. 소집일 **7일 전**까지 선수 및 소속팀에 서면 통보한다(KFA 국가대표팀 운영규정 제10조).

> **주의**: U-14 선발에 관한 세부 기준(선발 인원 수, 연간 소집 횟수)은 KFA가 공식 문서로 공개하지 않아 확인이 어렵다. 정확한 정보는 KFA(대표번호: 02-2002-0707) 또는 소속 시도 협회에 문의 권장.

## 3. U-15 / U-16

KFA는 **AFC U-17 아시안컵**(2년 주기)을 대비하여 대회 전년도에 U-15/U-16 대표팀을 운영한다.

| 항목 | 내용 |
|------|------|
| 주관 | KFA 기술위원회 / 해당 대표팀 감독 |
| 대상 | AFC U-17 아시안컵 기준 연령 (대회년도 기준 만 16세 이하) |
| 선발 방식 | 중학교 리그·K리그 U-18 아카데미 관찰 → 감독 추천 → 소집 훈련 캠프 |
| 선발 기준 | 학원 리그 성적, 물리적 조건, 포지션별 능력, 지도자 추천 |
| 소집 횟수 | 연 3~5회 (대회 준비 시기 집중) |
| 최종 엔트리 | AFC 예선 명단: 약 23명 |
| 주요 대회 | AFC U-17 아시안컵 예선 (2년 주기) |

**AFC U-17 아시안컵 예선 구조**: 예선 조별리그 → 각 조 상위팀이 본선 진출 → 본선 43개국 참가 (한국은 AFC 회원국으로 자동 예선 참가).

**최근 사례 (2025 AFC U-17 아시안컵 예선)**: 2008년생으로 구성된 U-16 대표팀 23명이 중국 다롄 예선에 참가하여 본선 진출을 확정했다.

> **참고**: KFA는 대회 연도에 따라 팀을 'U-15' 또는 'U-16'으로 혼용 표기한다. AFC 대회 규정상 출생연도 기준으로 연령을 산정한다.

## 4. U-17

| 항목 | 내용 |
|------|------|
| 주관 | KFA / 전임 대표팀 감독 |
| 대상 | 만 17세 이하 |
| 선발 방식 | 중학교·고교 리그 관찰 + U-16 대표팀 연속성 + 추가 캠프 |
| 선발 기준 | 대회 전략·전술 적합성, 최근 경기력, 신체 조건 |
| 소집 횟수 | 연 4~6회 이상 (대회 연도 집중) |
| 최종 엔트리 | AFC 본선·FIFA 월드컵: 21명 |
| 주요 대회 | AFC U-17 아시안컵, FIFA U-17 월드컵 |

### 주요 대회 사이클

| 대회 | 주기 | 출전 조건 |
|------|------|-----------|
| AFC U-17 아시안컵 | 2년마다 | 아시아 지역 예선 통과 |
| FIFA U-17 월드컵 | 2년마다 (AFC 아시안컵와 같은 해) | AFC 아시안컵 8강 이상 (2025년 기준 — 종전 4강에서 완화) |

**2025년 실적**:
- **AFC U-17 아시안컵 (사우디아라비아)**: 한국 4강 진출 → FIFA 월드컵 본선 진출권 확보
- **FIFA U-17 월드컵 카타르 2025**: 조별리그 F조 (멕시코 2-1 승, 스위스 0-0 무, 코트디부아르 3-1 승) → 조 2위로 32강 진출 → 32강 잉글랜드전 0-2 패배 탈락
- **감독**: 백기태 (포항 유소년 디렉터 출신)

> AFC U-17 아시안컵 8강 진출 시 FIFA U-17 월드컵 본선행이 보장되는 구조로 2025년부터 확대되어 아시아 티켓이 기존 4장에서 8장으로 늘었다.

## 5. U-20

| 항목 | 내용 |
|------|------|
| 주관 | KFA / 전임 대표팀 감독 |
| 대상 | 만 20세 이하 |
| 선발 방식 | K리그·대학 리그·해외 리그 관찰 → 소집 훈련 → 최종 명단 |
| 소집 횟수 | 연 3~5회 (국내 훈련 + 해외 평가전) |
| 최종 엔트리 | AFC 아시안컵·FIFA 월드컵: 21명 |
| 주요 대회 | AFC U-20 아시안컵, FIFA U-20 월드컵 |

### 주요 대회 사이클

| 대회 | 주기 | 출전 조건 |
|------|------|-----------|
| AFC U-20 아시안컵 | 2년마다 | 아시아 지역 예선 통과 |
| FIFA U-20 월드컵 | 2년마다 | AFC U-20 아시안컵 성적 기준 본선 티켓 배분 |

### 대한민국 역대 성적 (U-20)

- AFC U-20 아시안컵: **12회 우승** (역대 최다)
- FIFA U-20 월드컵: 2019년 준우승 (이강인 골든볼), 2023년 4강

**2025년 실적 (FIFA U-20 월드컵 칠레 2025)**:
- 이창원 감독 지휘
- 조별리그 1승 1무 1패 → 조 3위 (득실 0)로 16강 진출
- 16강 모로코전 1-2 패배 탈락
- 아시아 최초 U-20 월드컵 **4대회 연속 토너먼트 진출** 기록

**최종 명단 구성 과정 (2025 칠레 월드컵)**: AFC 아시안컵 주축 10명 유지 + 국내·해외 소집 훈련에서 두각을 나타낸 11명 추가. 2007년생 김현오가 연령보다 1년 어리게 월반 발탁.

## 6. U-23 (올림픽 대표팀)

| 항목 | 내용 |
|------|------|
| 주관 | KFA / 전임 대표팀 감독 |
| 대상 | 만 23세 이하 (올림픽 규정: 오버에이지 3명 추가 가능) |
| 선발 방식 | K리그·해외 리그 성적 + 감독 판단 |
| 최종 엔트리 | 올림픽 본선: 18명 (오버에이지 3명 포함) |
| 주요 대회 | AFC U-23 아시안컵 (올림픽 예선 겸용) |

### 올림픽 출전 구조

올림픽 남자 축구 아시아 본선 티켓은 **AFC U-23 아시안컵** 성적으로 배분된다.

| 올림픽 | AFC 티켓 수 | 비고 |
|--------|------------|------|
| 2024 파리 | 2장 | AFC U-23 아시안컵 준결승 진출 팀 |
| 2028 LA | 2장 | AFC U-23 아시안컵 결승 진출 팀 |

**2024 파리올림픽**: 한국은 AFC U-23 아시안컵(카타르, 2024.4) 8강에서 인도네시아에 승부차기(10-11) 패배 → **40년 만에 올림픽 본선 탈락** (1984년 LA 이후 최초). 황선홍 감독 지휘.

**2028 LA올림픽 준비**: 이민성 감독 체제로 2026 AFC U-23 아시안컵(올림픽 예선 겸용) 준비 중. 예선 1차전 5-0 대승 기록 (2025.9). 그러나 한국의 FIFA 랭킹 하락과 유럽 진출 선수 감소로 톱시드 확보 불투명 상태.

**군 면제**: 아시안게임 금메달 또는 올림픽 메달 수상 시 병역 특례 혜택. U-23 대표팀 합류 경력은 병역 이슈와 직결되어 선수 개인의 커리어에 큰 영향을 미친다.

## 7. 성인 A대표팀

| 항목 | 내용 |
|------|------|
| 주관 | KFA 기술위원회 / 성인 대표팀 감독 |
| 감독 | 홍명보 (2024년 7월 선임, 성인 2기) |
| 선발 방식 | 감독 전권 — 국내외 리그 관찰 후 감독이 최종 결정 |
| 선발 기준 | 최근 경기력, 소속팀 출전 시간, 전술 적합성, 부상 여부 |
| 소집 인원 | A매치 윈도우별 23~26명 |
| 연간 소집 횟수 | FIFA 국제 매칭 기간 기준 연 5~8회 (확인 필요) |
| 주요 대회 | FIFA 월드컵, AFC 아시안컵, 친선 A매치 |

### 최근 소집 사례

| 시기 | 인원 | 대회 |
|------|------|------|
| 2024년 6월 | 26명 | FIFA 월드컵 3차 예선 |
| 2025년 9월 | 26명 | FIFA 월드컵 3차 예선 |
| 2026년 3월 | 23명 | FIFA 월드컵 최종 예선 |

### 감독 체계

- 대표팀 감독은 **KFA 전략강화위원회 또는 기술발전위원회 추천** → **이사회 임명** 절차를 거친다.
- 선발된 감독은 소집 계획을 **국가대표전력강화위원회**에 제출해야 하며, 소집일 **7일 전**까지 선수·소속팀에 서면 통보한다.
- 국내파와 해외파 모두 대상. 경기력이 인정되면 K리그 선수도 발탁된다.

**2026 FIFA 월드컵 (북중미)**: 한국은 아시아 3차 예선을 통해 본선 진출을 목표로 하고 있다 (2026.6 개막).

## 8. 여자 국가대표팀 (간략)

| 대표팀 | 주요 대회 | 비고 |
|--------|-----------|------|
| 여자 U-17 | AFC U-17 여자 아시안컵, FIFA U-17 여자 월드컵 | 한국은 2010년 FIFA U-17 여자 월드컵 우승 |
| 여자 U-20 | AFC U-20 여자 아시안컵, FIFA U-20 여자 월드컵 | U-20이 성인 전 마지막 연령별 대표팀 |
| 여자 성인 | FIFA 여자 월드컵, AFC 여자 아시안컵 | 올림픽 여자 축구는 U-23 제한 없음 |

- 여자 축구는 올림픽에 연령 제한이 없어 **U-23 별도 팀이 없다**.
- 최근 주목 선수: 케이시 유진 페어 (한국계 미국인, 대표팀 자격 선택), 천가람 등.
- 선발 절차는 남자와 동일하게 KFA 기술위원회·전임 감독이 주관한다.

## 9. KFA 골든에이지 아카데미 시스템

골든에이지는 2014년부터 KFA가 시행한 **유소년 체계적 육성 프로그램**이다.

### 프로그램 구조

```
프리 골든에이지 (6~11세) → 골든에이지 (12~15세) → 포스트 골든에이지 (16~19세)
```

### 운영 방식

- **지역센터(21개)**: 시도 축구협회 운영. 지역 학교 팀·클럽 선수 대상 기초 발굴
- **광역센터(5개)**: KFA + 시도 협회 공동 운영. 지역 유망주 집중 훈련
- **합동광역센터**: KFA 주관. 광역센터 우수 선수 선발
- **KFA센터**: 최상위. 국가대표급 유망주 집중 관리

### 골든에이지 대표팀 선발

'KFA 골든에이지 대표팀'은 **8~15세 유망주 중 동연령대 국내 최고 수준** 선수로 구성된다.

2025년 선발 사례: 박건민, 이동관, 김정음, 박준범 등이 'KFA 골든에이지 대표팀'에 선발된 바 있다.

### AFC 평가

KFA는 AFC '엘리트 유소년 축구 발전 계획' 프로젝트에서 11개 항목(조직·계획·경기·지도자·유망주 발굴·의료·스포츠과학·교육·시설·철학·클럽 유소년 아카데미) 평가를 통해 **3스타 정규 멤버십**을 유지하고 있다 (2019년 프로젝트 시작, 2020년 수상).

## 10. 해외파 선수의 국대 참가 규정

### FIFA 기본 자격 요건

한국 국가대표팀에 출전하려면 원칙적으로 **대한민국 국적**을 보유해야 한다.

| 조건 | 내용 |
|------|------|
| 국적 | 대한민국 국적 (이중국적 포함) |
| 연령별 대표 참가 | 연령 조건 충족 시 국적만 있으면 가능 |
| A매치 출전 후 국적 변경 | 원칙적으로 불가 |

### FIFA 국적 변경 규정 (2021년 개정)

2020년 9월 FIFA 규정 개정 이후:
- **만 21세 이전에 A매치 3경기 이하** 출전한 경우 → 다른 국가 대표팀으로 변경 가능
- **월드컵 본선 또는 대륙별 국가대항전 본선 출전 경력** → 국적 변경 불가 (영구 고정)
- 국적 추가(귀화) 후 해당 국가에 18세 이후 **5년 이상 계속 거주** 요건 필요

### 해외 진출 선수 소집 절차

- 해외 클럽 소속 선수도 KFA가 소집장을 발송하면 **클럽은 원칙적으로 방출 의무**가 있다 (FIFA 국제 소집 윈도우 기간 한정).
- 연령별 대표팀(U-17, U-20 등)의 경우 해외 클럽과 협의 하에 소집 — 클럽이 거부하는 경우도 있어 **실제 합류 여부는 클럽 허가에 달려 있다**.

**사례**: 2025 FIFA U-20 월드컵 칠레 최종 명단 발표 시 스토크시티 배준호, 토트넘 양민혁 등 유럽파의 합류가 **클럽 측 거부로 무산**된 바 있다.

## 11. 국대 경력이 진로에 미치는 영향

### 프로 스카우트에 미치는 영향

- 연령별 국가대표 경력은 **스카우트의 직접 관찰 기회**를 높인다. 국제 대회에서 활약하면 해외 클럽의 관심을 받는 가장 빠른 경로가 된다.
- K리그 구단들도 연령별 대표팀 활동을 중요한 선발 기준 중 하나로 활용한다.

### 대학 진학에 미치는 영향

- 국가대표 경력은 대학 체육특기자 수시 지원 시 **실적 가산**이 된다.
- 단, 축구 선수의 경우 프로 지망 시 대학 진학이 필수가 아니며, K리그 산하 U-18 아카데미 졸업 후 프로 입단하는 경우도 많다.

### 병역 혜택과의 연계

- U-23 대표팀 → 아시안게임 금메달 시 **병역 특례** 가능 (사실상 선수 커리어에 결정적 요소).
- 올림픽 메달(동메달 이상)도 병역 특례 대상.

### 확률 참고

| 단계 | 통과 확률 (추정) |
|------|----------------|
| 초등 → 중학 → 고교 → 대학 축구부 진학 | 약 26% |
| K리그 선수 → 국가대표 발탁 | K리그 650명 중 약 3% |
| 초등 시작 → 성인 국가대표까지 | 약 0.38% |

> 출처: 클리앙 통계 자료 (확인 필요 — 공식 KFA 통계 아님)

## 12. 초등 6학년이 국대를 목표로 한다면 — 현실적 로드맵

### 경로 요약

```
초등 6학년
    ↓ (현재)
[지역 학원·클럽팀 등록 + KFA 골든에이지 지역센터 참여]
    ↓
중학교 입학 (만 13~15세) — 골든에이지 단계
    ↓
[학교 축구부 or K리그 U-15 클럽 아카데미 입단]
    ↓
KFA U-14 골든에이지 대표팀 추천 선발 가능 구간
    ↓
중학교 3학년~고등학교 1학년 (만 15~16세)
    ↓
[U-15/U-16 대표팀 → AFC U-17 아시안컵 예선 참가]
    ↓
고등학교 (만 17~19세) — 포스트 골든에이지 단계
    ↓
[U-17 대표팀 → AFC U-17 아시안컵·FIFA U-17 월드컵]
    ↓
[K리그 프로 데뷔 or 해외 진출 or 대학 진학]
    ↓
[U-20 대표팀 → AFC U-20 아시안컵·FIFA U-20 월드컵]
    ↓
[U-23 대표팀 → AFC U-23 아시안컵(올림픽 예선)]
    ↓
성인 A대표팀
```

### 단계별 핵심 포인트

| 단계 | 연령 | 핵심 목표 | 주의사항 |
|------|------|-----------|----------|
| 초등 6학년 (현재) | 만 12세 | 기본기 완성, 소속팀 리그 성적 | KFA 지역센터 참여 여부 확인 |
| 중1~중2 | 13~14세 | U-14 골든에이지 대표팀 추천 | 학교 팀 or K리그 U-15 아카데미 선택 중요 |
| 중3~고1 | 15~16세 | U-15/U-16 대표팀 소집 | AFC U-17 예선 준비 사이클에 맞춰야 |
| 고2~고3 | 17~18세 | U-17 대표팀, 프로 계약·해외 진출 | 이 시기 두각이 없으면 국대 경로 매우 어려워짐 |
| 대학·프로 초반 | 19~20세 | U-20 대표팀 | 대학 vs 프로 입단 선택 결정 필요 |
| 프로 중반 | 22~23세 | U-23 올림픽 대표팀 | 병역 문제 해결 여부와 연결 |
| 프로 전성기 | 24세 이상 | 성인 A대표팀 | 꾸준한 K리그·해외 출전 시간 필수 |

### 현실적 조언

1. **K리그 클럽 U-15 아카데미 입단이 최선**: 중학교 진학 시 학교 팀보다 K리그 산하 아카데미(예: 전남, 수원, 인천, 울산 등)가 KFA 스카우트의 시야에 들어올 가능성이 높다.
2. **KFA 골든에이지 지역센터 참여**: 지역센터 등록만으로도 KFA 지도자의 관찰을 받을 수 있다.
3. **국제 대회 출전 이력**: U-14 단계부터 국제 친선 경기 경험을 쌓으면 이후 연령별 대표팀 선발에서 유리하다.
4. **성적보다 성장세**: KFA 선발 기준은 현재 성적뿐 아니라 **성장 가능성**을 중시한다는 점이 여러 지도자 인터뷰에서 확인된다.
5. **해외 진출**: 이강인(발렌시아 유소년), 양민혁(강원FC → 토트넘) 사례에서 보듯 국내 아카데미에서 두각을 나타낸 뒤 해외 진출 → 연령별 국가대표 경로가 가장 이상적인 모델이다.

### 참고 사례

| 선수 | 유소년 경로 | 국대 진입 시점 |
|------|------------|---------------|
| 이강인 | 발렌시아 유소년 (한국 초등 후 스페인 직행) | U-17부터 연령별 대표 → 2019 U-20 월드컵 골든볼 |
| 양민혁 | 강원FC 아카데미 → K리그 고등학생 데뷔 | 18세 132일, 역대 최연소 13번째 A매치 데뷔 (2024) |
| 이승우 | 바르셀로나 유소년 (초등 후 스페인 직행) | U-17부터 연령별 대표 경력 |
