## 1. Executive Summary

### 핵심 발견사항

1. **클럽팀의 부상**: 2025년 전국 중등축구리그 왕중왕전에서 클럽팀(충남천안축구센터)이 **5관왕**을 달성. 4강 중 2팀이 클럽팀. 더 이상 학원축구만이 유일한 선택지가 아님.

2. **K리그 유스 출신 프로 비율 급증**: 2018년 25.7% → 2025년 약 39%. 매년 3~4%p씩 증가하며 학원축구를 빠르게 역전 중.

3. **학원축구의 구조적 리스크**: 불법 찬조금 관행 여전(2025년 인천 고교 축구부 수사 사례), 비용 불투명성 높음.

4. **월드컵FC 판단 핵심**: 공개 정보가 제한적. **반드시 직접 방문하여 KFA 등록 여부, 대회 성적, 졸업생 진학 실적을 확인**해야 함.

### 최종 추천 (1/2/3순위)

| 순위 | 추천 경로 | 이유 |
|:---:|---|---|
| **1순위** | **K리그 유스 U-15** (FC서울/수원삼성/울산HD) | 비용 제로, 프로 직행, 체계적 육성. 입단 가능하다면 최선 |
| **2순위** | **검증된 클럽팀** (천안축구센터급) 또는 **월드컵FC** (실적 확인 후) | 학업 병행 + 진로 유연성. 단, 팀 퀄리티 직접 검증 필수 |
| **3순위** | **학원축구 명문** (문래중/목동중) | 안정적 학교생활 + 고교 네트워크. 비용/찬조금 리스크 감수 필요 |

## 2. 대회 성적 및 팀 데이터 분석

### 2-1. 대한민국 중등 축구 주요 대회 체계 (2025-2026 기준 상세)

#### A. 연중 리그 (3~10월)

**1) 전국 중등 축구리그 (권역별)**
- **주최/주관**: 대한축구협회(KFA) 주최, 각 시도 축구협회 주관
- **시기**: 3월~10월 (매주 주말, 7월 방학기간 일시 중단 후 9월 재개)
- **참가 자격**: KFA 등록 학원팀(중학교 축구부) + 클럽팀 + K리그 유스팀 전체
- **형식**: 권역별 풀리그 (홈&어웨이). U-15(고학년부)와 U-14(저학년부) 분리 운영. 저학년부는 스플릿 리그 추가 진행
- **권역 구성**: 서울/인천, 경기(수원/용인/광명 등 세분화), 강원, 충청, 대전/세종, 전북, 전남/광주, 대구/경북, 부산/울산/경남, 제주 등 약 15개 권역
- **참가팀 수**: 전국 약 200여 팀 (권역당 8~15개 팀)
- **개최지**: 각 팀 홈 구장 및 지역 체육시설
- **최근 권역 우승**: 2025 강원권역 U-15 홍천FC 우승

**2) K리그 주니어 U-15 리그**
- **주최/주관**: 한국프로축구연맹(K리그) / KFA
- **시기**: 3월~10월 (권역별 리그와 병행)
- **참가 자격**: K리그 1부/2부 산하 U-15 유소년팀 한정 (약 25개 팀)
- **형식**: A권역(중부)·B권역(남부) 각 11~13개 팀, 2라운드 로빈 총 20~24R 리그전. 권역 우승팀 결정
- **참가팀 수**: 약 25개 팀
- **개최지**: 각 구단 홈 경기장 및 지역 인조잔디 구장
- **최근 우승팀**:

| 연도 | U-15 우승 | 비고 |
|:---:|-----------|------|
| 2022 | 수원삼성 U-15 | 2연패 |
| 2023 | 수원삼성 U-15 | 3연패 |
| 2024 | 수원삼성 U-15 | 4연패, 21경기 무패(18승3무) |
| 2025 | 확인 필요 | - |

#### B. 리그 왕중왕전 (11월)

**3) 전국 중등 축구리그 왕중왕전**
- **주최/주관**: KFA 주최, 개최지 축구협회 주관
- **시기**: 11월 (약 12일간)
- **참가 자격**: 전국 권역별 리그 상위 팀 (각 권역 1~2위), 총 64개 팀
- **형식**: 64강 단판 토너먼트 (예선 조별리그 후 본선 단판 녹아웃)
- **참가팀 수**: 64개 팀
- **개최지**: 2025년 경북 영덕 / 매년 유치 경쟁
- **최근 우승팀**:

| 연도 | 우승 | 준우승 | 비고 |
|:---:|------|--------|------|
| 2025 | 충남천안축구센터 U-15 | 서울 목동중 | 결승 3-2 |

#### C. K리그 유스 챔피언십 (8월)

**4) K리그 유스 챔피언십 U-15 (= GROUND.N K리그 U-15 챔피언십)**
- **주최/주관**: 한국프로축구연맹(K리그)
- **시기**: 8월 (약 12일간, 여름방학 중)
- **참가 자격**: K리그 산하 U-15 유소년팀 전체 (약 25개 팀)
- **형식**: 조별 리그 + 16강부터 단판 토너먼트
- **참가팀 수**: 약 25개 팀
- **개최지**: 2024·2025년 충남 천안시 천안축구센터

| 연도 | U-15 우승 | U-14 우승 | 비고 |

#### D. 전국 토너먼트 대회 (연간 5~6개)

| **추천 학교** | 문래중(후임 감독 확인 후), 목동중, 백마중(경기 지역) |

### 10-4. 최종 권고: 병행 전략

> **가장 현명한 접근은 "병행 전략"입니다.**

| 단계 | 행동 | 시기 |
|:---:|------|------|
| **1단계** | 월드컵FC 직접 방문, 훈련 참관, 감독 면담 (체크리스트 기반) | **즉시** |
| **2단계** | K리그 유스 셀렉션 일정 확인 및 지원 (FC서울/수원삼성/인천 등) | **2026년 11~12월** |
| **3단계** | 학원축구 명문 입단 테스트도 병행 탐색 | **2026년 11~2027년 1월** |
| **4단계** | 모든 결과를 종합하여 최종 결정 | **2027년 1~2월** |

**우선순위 정리:**
1. K리그 유스 합격 → **즉시 수락** (최선의 선택)
2. K리그 유스 불합격 + 월드컵FC 검증 통과 → **월드컵FC 수락** (차선)
3. K리그 유스 불합격 + 월드컵FC 검증 미달 → **학원축구 명문** (대안)

## 11. 부모님 면담 시 압박 질문 리스트 10가지

감독/코치를 만날 때 반드시 물어야 할 질문입니다. 질문의 의도와 좋은/나쁜 답변 기준을 함께 제시합니다.

### 질문 1: "최근 3년간 졸업생들이 어떤 고등학교에 진학했습니까? 구체적인 학교 이름과 인원을 알려주십시오."

| 항목 | 내용 |
|------|------|
| **의도** | 팀의 실질적 진학 네트워크와 실적을 확인 |
| **좋은 답변** | 구체적 학교명+인원 즉시 제시. "작년 A군은 포항제철고, B군은 매탄고..." |
| **나쁜 답변** | 모호한 답변("다들 좋은 데 갔어요"), 구체적 데이터 없음 |

### 질문 2: "감독님의 AFC 지도자 라이선스 등급은 무엇이며, 코치진 전원의 자격증을 확인할 수 있습니까?"

| 항목 | 내용 |
|------|------|
| **의도** | 코칭스태프의 공식 자격 검증 |
| **좋은 답변** | AFC B/A 라이선스 이상. 자격증 사본 즉시 제공 |
| **나쁜 답변** | 자격증 언급 회피, "경험이 중요하지 자격증이 중요한 게 아니다" |

### 질문 3: "월 회비 외에 연간 발생하는 모든 추가 비용을 항목별로 서면 제공해주실 수 있습니까?"

| 항목 | 내용 |
|------|------|
| **의도** | 숨겨진 비용(찬조금, 전지훈련비 등) 사전 파악 |
| **좋은 답변** | 항목별 비용 명세서 즉시 제공. "연간 총 비용은 약 OOO만원입니다" |
| **나쁜 답변** | "그때그때 달라요", "학부모회에서 알아서 해요" |

### 질문 4: "선수가 부상을 당했을 때 치료 및 재활은 어떻게 진행됩니까? 보험 가입 현황은?"

| 항목 | 내용 |
|------|------|
| **의도** | 부상 관리 시스템과 보험 체계 확인 |
| **좋은 답변** | 협약 병원 명시, 스포츠안전공제 가입 확인, 재활 프로그램 설명 |
| **나쁜 답변** | "크게 다친 애가 없어서...", 보험 가입 여부 불명확 |

### 질문 5: "팀에 입단한 선수 중 중도 이탈한 비율은 어떻게 됩니까? 이탈 사유는 주로 무엇입니까?"

| 항목 | 내용 |
|------|------|
| **의도** | 팀 내부의 분위기와 선수 관리 실태 파악 |
| **좋은 답변** | 솔직한 수치 제시. "작년 20명 중 2명이 전학. 사유는 가정 사정" |
| **나쁜 답변** | 이탈률 언급 회피, "우리 팀은 한 번 오면 안 나가요" (비현실적) |

### 질문 6: "훈련 중 체벌이나 폭언에 대한 팀 규정은 무엇입니까? 과거 관련 이슈가 있었습니까?"

| 항목 | 내용 |
|------|------|
| **의도** | 선수 인권 보호 체계 확인 |
| **좋은 답변** | 명문화된 규정 제시, 인권교육 시행 언급, 과거 이슈 투명 공개 |
| **나쁜 답변** | 불쾌한 반응, "요즘 세상에 누가 그래요" (구체적 체계 없음) |

### 질문 7: "이 팀이 KFA에 정식 등록되어 있습니까? 전국 중등 축구리그에 참가하고 있습니까?"

| 항목 | 내용 |
|------|------|
| **의도** | 공식 대회 참가 자격과 선수 경력 인정 여부 확인 |
| **좋은 답변** | KFA 등록증 즉시 제시, 리그 참가 이력 공개 |
| **나쁜 답변** | 등록 여부 애매, "친선전 위주로 해요" (공식 리그 미참가) |

### 질문 8: "감독님이 이 팀을 떠나시게 되면, 선수들의 진학과 훈련은 어떻게 보장됩니까?"

| 항목 | 내용 |
|------|------|
| **의도** | 감독 개인 의존도와 팀 시스템의 지속가능성 파악 |
| **좋은 답변** | 코칭 시스템과 운영 체계가 감독 개인에 의존하지 않음을 설명 |
| **나쁜 답변** | 답변 회피, 불쾌한 반응 (= 감독 개인에 과도하게 의존하는 구조) |

### 질문 9: "선수가 3학년 때 K리그 유스나 다른 고교로 진학하고 싶을 경우, 이적에 제한이 있습니까? 위약금이나 계약 조건이 있습니까?"

| 항목 | 내용 |
|------|------|
| **의도** | 선수의 진로 자유도와 팀의 구속 조건 확인 |
| **좋은 답변** | "제한 없습니다. 아이의 미래가 최우선입니다" + 서면 확인 |
| **나쁜 답변** | 위약금 언급, "우리 팀에서 키운 선수인데..." (구속 의도) |

### 질문 10: "기존 학부모님 2~3분과 직접 이야기할 수 있도록 연결해주실 수 있습니까?"

| 항목 | 내용 |
|------|------|
| **의도** | 팀의 실체를 기존 구성원을 통해 교차 검증 |
| **좋은 답변** | 즉시 연락처 제공. "OO어머니, OO아버지께 연락해보세요" |
| **나쁜 답변** | 거부하거나 특정 학부모만 연결 (= 사전 조율된 답변 가능성) |

> **TIP**: 질문 10이 가장 중요합니다. 학부모 연결을 거부하는 팀은 **즉시 제외**하십시오. 또한, 팀에서 연결해준 학부모 외에 **별도로** 기존/퇴단 학부모를 찾아 이야기하는 것이 가장 객관적인 정보를 얻는 방법입니다.
